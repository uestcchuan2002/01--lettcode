# 283.移动零
## 暴力解法
```cpp
void moveZeroes(vector<int>& nums) {
        for (int i = 0; i < nums.size(); i++) {
            if (nums[i] != 0) {
                for (int j = 0; j < i; j++) {
                    if (nums[j] == 0) {
                        nums[j] = nums[i];
                        nums[i] = 0;
                    }
                }
            }
        }
    }
```

## 双指针
**right指针** 遍历整个容器
**left指针** 指向已经处理好的非零数据的末尾
```cpp
void moveZeroes(vector<int>& nums) {
    // 双指针
    int right = 0;
    int left = 0;
    while (right < nums.size()) {
        if (nums[right] != 0) {
            swap(nums[right], nums[left]);
            left++;
        }
        right++;
    }
}
```
